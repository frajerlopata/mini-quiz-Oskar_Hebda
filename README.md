# Mini Quiz App

## Opis projektu
Aplikacja quizowa stworzona w Android Studio. Użytkownik odpowiada na pytania i otrzymuje wynik końcowy.

## Instrukcja uruchomienia
1. Pobierz repozytorium
2. Otwórz w Android Studio
3. Poczekaj na synchronizację Gradle
4. Kliknij Run (▶)
